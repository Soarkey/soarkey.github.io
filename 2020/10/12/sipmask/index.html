Abstract

​ Our main contribution is a novel light-weight spatial preservation (SP) module that generates a separate set of spatial coefficients for each sub-region within a bounding-box, leading to improved mask predictions.

mask alignment weighting loss + feature alignment scheme

1. Introduction

**发展：**TensorMask(精度和two-stage是comparable，但速度比two-stage的Mask R-CNN还慢) ➡️ YOLACT(33fps，但精度低于two-stage)

**原因：**YOLACT损失了空间信息，spatial information within an object (bounding-box)，归因于the utilization of a single set of object-aware coeﬃcients to predict the whole mask of an object，本文加入了SP模块，能够预测object-aware spatial coefficiens，从而将mask预测划分为对多个sub-mask预测，提高对空间相邻目标的描绘能力。

two-stage: Mask R-CNN(RPN+RoIAlign) ➡️ FCIS/PANet(PSRoI+FPN) ➡️ MS R-CNN(additional mask-IoU)

ont-stage: YOLACT(basis masks+single set instance-specific coefs <for each bbox>) ➡️ SipMask (SP module + multi sets … <吸收YOLCAT的优点，改进缺点>)

2. Method

2.1 Architecture

​ 基于FCOS的anchor-free架构，替换FCOS中的标准cls/reg为mask-specialized reg/cls brances，两个分支均为全卷积结构。

​ backbone为FPN，将bbox划分为k×kk\times kk×k的sub-regions计算每个sub-region空间系数的separate set。

​ （SMP为spatial mask prediction，CONVS为3x3卷积）

【Spatial Preservation (SP) Module + Mask-specialized Regression Branch + Spatial Mask Prediction (SMP) Module 】

2.2 Spatial Preservation (SP) Module

two task： spatial coeﬃcients generation and feature alignment

  • The spatial coeﬃcients are introduced to improve mask prediction by preserving spatial information within a bounding-box.

  • Our feature alignment scheme aims at improving the feature representation for both box classiﬁcation and spatial coeﬃcients generation.

  • Spatial Coeﬃcients Generation

  • Feature Alignment Scheme： 用了可变形卷积，输入为reg offset， y(p0)=∑i∈Gwr⋅x(p0+pr+Δpr)y(p_0)=\sum_{i\in G} w_r \cdot x(p_0+p_r+\Delta p_r)y(p0​)=∑i∈G​wr​⋅x(p0​+pr​+Δpr​)
